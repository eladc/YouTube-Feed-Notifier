YouTube-Feed-Notifier (API v2)
==============================

Pop-up notifications about new videos from YouTubers you are subscribed to.

YTFN is set to check for new videos every 15 minutes.

Place `youtube.png` in

    /usr/share/pixmaps

Usage:
-------
    main.py username
